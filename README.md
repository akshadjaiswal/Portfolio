## Getting Started

To view the website locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/akshadjaiswal/Portfolio.git
    ```

2. Navigate into the project directory:
    ```bash
    cd Portfolio
    ```

3. Open the `index.html` file in your browser.

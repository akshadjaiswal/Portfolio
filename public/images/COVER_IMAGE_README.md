# Cover Image Instructions

## What to add

Place your cover image here as: **`cover.jpg`**

## Specifications

- **File name**: `cover.jpg` (or update the path in ProfileHeader.tsx)
- **Recommended size**: 1920x1080px or higher
- **Aspect ratio**: 16:9 or wider landscape
- **Format**: JPG, PNG, or WebP
- **Theme**: Nature, landscape, mountains, cityscape, or any inspiring background
- **Quality**: High resolution for sharp display

## Tips

1. Choose an image with good contrast so the white quote text is readable
2. Landscape/nature photos work best
3. Avoid busy images - simpler backgrounds look cleaner
4. The gradient overlay will be applied automatically for text readability

## Example Sources

- Unsplash (https://unsplash.com) - Free high-quality photos
- Pexels (https://pexels.com) - Free stock photos
- Your own photography

## Current Setup

The component will show a 404 error until you add `cover.jpg` to this directory.
The layout and functionality will work perfectly once the image is added!
